# OneKash Labs

**Open Source. Modern. Your tools. Your data. Your rules.**

Privacy-focused apps and developer tools. No tracking. No accounts. No nonsense.

## Apps

**[KashCal](https://kashcal.github.io)** - A modern calendar that works offline and syncs with iCloud.
[GitHub](https://github.com/KashCal/KashCal) · [F-Droid](https://f-droid.org/en/packages/org.onekash.kashcal/) · [Download](https://github.com/KashCal/KashCal/releases/latest)

**[KashCake](https://github.com/KashCal/KashCake)** - Never forget a birthday. Syncs with contacts, stays on your device.
[Download](https://github.com/KashCal/KashCake/releases/latest)

## Developer Tools

**[iCloud Calendar MCP Server](https://github.com/icloud-calendar-mcp/icloud-calendar-mcp)** - Access iCloud Calendar from Claude Code, Cursor, and other AI tools.
[npm](https://www.npmjs.com/package/@icloud-calendar-mcp/server) · [PyPI](https://pypi.org/project/icloud-calendar-mcp/)

**[iCalDAV](https://github.com/iCalDAV/iCalDAV)** - Kotlin CalDAV client with offline sync and conflict resolution. Powers KashCal's calendar sync.
[Website](https://icaldav.github.io)

## Philosophy

**Your data is yours.** It lives on your device. We never see it, store it, or sell it.

**Open by default.** All our code is public. See for yourself.

**Design matters.** Beautiful interfaces aren't just for Big Tech.

**Does one thing well.** Each app has one job. It does it well.
